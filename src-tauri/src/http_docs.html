观复 API

文献论争图谱工具 — HTTP API — localhost:17800

AI Agent 集成端点：GET /openapi.json 返回完整的 OpenAPI 3.0 规范。
所有请求使用 JSON 格式。仅监听 127.0.0.1，不对外暴露。

系统

GET/health
健康检查
GET/openapi.json
OpenAPI 3.0 规范（供 AI Agent 使用）AI

项目

POST/api/projects/create
新建项目 — { "name": "...", "baseDir": "/path" }
POST/api/projects/open
打开项目 — { "projectPath": "/path/proj.guanfu" }
POST/api/projects/save
保存项目 — { "project": {...} }
GET/api/projects/recent
获取最近项目路径
GET/api/projects/default-dir
获取默认项目目录
GET/api/projects/info?projectPath=...
获取项目完整信息（文献 + 关系 + 布局）AI

文献

POST/api/papers/import
导入 PDF — { "projectPath": "...", "filePaths": ["/a.pdf"] }
GET/api/papers/list?projectPath=...
列出项目中的所有文献
DELETE/api/papers/{id}?projectPath=...
删除文献
POST/api/papers/{id}/update
更新文献信息 — { "projectPath": "...", "paper": {...} }
POST/api/papers/{id}/ai-parse
AI 解析文献 — { "projectPath": "..." }
POST/api/papers/{id}/extract-text
提取 PDF 全文文本（含 OCR）AI
POST/api/papers/batch-ai-parse
批量 AI 解析 — { "projectPath": "...", "paperIds": ["id1", "id2"] }AI

示例

# 打开项目并获取信息
curl -X POST http://localhost:17800/api/projects/open \
  -H "Content-Type: application/json" \
  -d '{"projectPath": "/path/to/MyProject.guanfu"}'

# 导入 PDF
curl -X POST http://localhost:17800/api/papers/import \
  -H "Content-Type: application/json" \
  -d '{"projectPath": "/path/p.guanfu", "filePaths": ["/tmp/paper1.pdf"]}'

# 列出文献
curl "http://localhost:17800/api/papers/list?projectPath=/path/to/MyProject.guanfu"

# 批量 AI 解析
curl -X POST http://localhost:17800/api/papers/batch-ai-parse \
  -H "Content-Type: application/json" \
  -d '{"projectPath": "/path/p.guanfu", "paperIds": ["id1", "id2"]}'

# 推荐关系
curl -X POST http://localhost:17800/api/ai/recommend-relations \
  -H "Content-Type: application/json" \
  -d '{"projectPath": "/path/to/MyProject.guanfu"}'

# 运行洞察分析
curl -X POST http://localhost:17800/api/insights/run \
  -H "Content-Type: application/json" \
  -d '{"projectPath": "/path/to/MyProject.guanfu"}'

# 知识库对话
curl -X POST http://localhost:17800/api/chat/ask \
  -H "Content-Type: application/json" \
  -d '{"projectPath": "/path/p.guanfu", "question": "研究方法有哪些趋势？"}'
